Onitama: Pokémon Edition

Voor dit project hebben we de klassieke Onitama-mechanics gecombineerd met de wereld van Pokémon. In plaats van standaard pionnen en kaarten, hebben we een custom interface gebouwd waarin spelers met hun favoriete Pokémon strategische zetten uitvoeren in een digitale arena.

Custom Visuals & Logic

Naast de complexe C# backend voor de spelregels en zet-validatie, hebben we veel aandacht besteed aan de frontend beleving. De kaarten zijn volledig aangepast met Pokémon-artwork en de grid-interacties voelen aan als een echte battle. Hoewel de live-verbinding met de API momenteel niet actief is, toont de code onze vaardigheden in Object Oriented Programming en dynamische UI-updates.
